Thank you for installing Nene bot!

Nene bot was developed by Uzair Ashraf. It was originally created to be used as an open source api to serve image links pointing to anime reactions. It is now a Slack bot and may also be used as an api in your projects. The project is completely open source and you may add your own images if you would like.
To get started using Nene, type /nene info in Slack to see which commands are available!
Want to contribute your own reactions to this project? Find the repo on Github!
